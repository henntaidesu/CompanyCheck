§1

公司是做什么的

渡辺電機工業株式会社是一家专注于「计测・制御机器」的专业制造商，创业86年（创业1940年11月3日，法人设立1951年7月3日）。总部位于东京都渋谷区神宮前，另在大阪、名古屋设营业所，调布与福岛设有工场。

技术上整合了计测技术、エレクトロニクス技术、メカトロニクス技术与ソフトウェア技术，从产品开发做到售后客户支持。近年主打产品线包括 IoT ゲートウェイ／远程监控单元、电力量计・デマンド计等，公司正朝工业IoT方向转型。

Solution

ソリューションビジネス — 电力・能源监控系统、大楼（ビル）监控系统

Component

コンポーネントビジネス — 信号变换器、模拟/数字面板仪表、温度传感器、リモートI/O

Automotive

自动车整备机器业务 — 汽车维修检测设备

一句话总结：这是一家中小规模、历史悠久（近90年）的 B2B 工业计测仪表制造商，不是消费电子或互联网公司，客户是工厂、大楼设施管理方、汽车维修厂等。

§3

岗位详情：設計開発エンジニア（ハードウェア・ソフトウェア）

该公司「设计开发」岗位覆盖硬件与软件两条线（新卒采用学科要求「理系全学部全学科」），本质是面向公司主力产品——计测仪表 / 电力监控模块 / IoT网关——的软硬件一体化产品开发工程师。

硬件方向

  • 回路设计 — 模拟电路、数字电路、单片机周边电路
  • 基板・机构设计 — 筐体与结构设计
  • 性能评价 — 试作品动作确认、性能测试
  • EMC对策 — 公司具备完整EMC测试环境
  • 量产引継 — 组装作业工序书、检查规格书编写

软件方向

  • 嵌入式开发 — Linux Boot 底层开发（含 BSP 板级支持包）
  • WEB・クラウド开发 — IoT网关产品的云端连接功能
  • 应用程序开发 — 运行于 Linux 的应用软件，服务 IoT Gateway 产品线

符合公司「计测技术 + 电子 + 机电一体 + 软件」四位一体的整合战略，也呼应官网强调的产品向 IoT 化转型方向。入职后配有导师制（每位新人对接一名资深前辈一对一带教）。
